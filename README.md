# Simulation-of-Ship-Motion-and-Ocean-Dynamics-in-Storm-Conditions-using-Blender
This project focuses on the simulation of ship motion and ocean dynamics during extreme storm conditions using Blender, with integration possibilities for Augmented Reality (AR) and Virtual Reality (VR) environments.
The system recreates a highly immersive maritime scenario where a pirate ship navigates through a violent storm at sea, demonstrating realistic ocean physics, weather effects, and ship motion dynamics.


🎯 Main Objectives

1.Design and simulate a realistic storm environment including:
Ocean waves
Rain
Wind
Lightning

2.Model and animate a pirate ship reacting naturally to:
High waves
Strong winds

3.Simulate ship motion dynamics:
Pitch
Roll
Yaw

4.Create dynamic weather effects:
Heavy rain particles
Dark storm clouds
Lightning flashes

5.Provide an immersive AR/VR experience of ship behavior during extreme storm conditions.

6.Demonstrate the application of 3D simulation and AR/VR technologies in:
Maritime visualization
Training scenarios

7.Improve understanding of ocean–ship interaction under severe weather conditions.

8.Integrate realistic storm sound effects for enhanced immersion.


🛠️ Technologies Used

Blender (Modeling, Animation, Physics Simulation, Rendering)
Particle System (Rain Simulation)
Ocean Modifier (Wave Simulation)
Physics & Rigid Body Dynamics
Volumetric Shaders (Storm Clouds)
Emission Shaders (Lightning Effects)
AR/VR Integration (Export-ready for immersive environments)
Sound Design Integration



🌪️ Simulation Features

🌊 Ocean Dynamics
Realistic storm-driven waves using Blender’s Ocean Modifier
Adjustable wave height, wind speed, and choppiness
Physically responsive ship–wave interaction

🚢 Ship Motion Dynamics
Realistic ship movement:
Pitch (forward/backward tilt)
Roll (side-to-side tilt)
Yaw (rotational movement)
Natural response to rough sea conditions

🌧️ Weather Effects
Heavy rainfall using particle systems
Volumetric storm clouds
Dark cinematic lighting
Animated lightning flashes with emission shaders

🔊 Sound Effects
Thunder sounds
Heavy rain ambience
Storm wind effects
Ocean wave sounds



▶️ How to Run the Project

1.Install Blender (Latest Version Recommended)

2.Open the storm_simulation.blend file

3.Adjust simulation parameters if required:
Ocean modifier settings
Particle count
Lighting intensity

4.Render animation using:
Eevee (Faster preview)
Cycles (High realism)



🎥 Rendering Settings (Recommended)

Engine: Cycles
Samples: 128–300
Motion Blur: Enabled
Volumetrics: Enabled
Frame Rate: 24 fps
